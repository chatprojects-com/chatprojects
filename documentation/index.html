Getting Started

Up and running in 5 minutes or less.

1

Install Plugin

Upload to /wp-content/plugins/ and activate via Plugins menu.

2

Add API Key

Go to ChatProjects > Settings and add at least one API key.

3

Create Page

Add [chatprojects_main] shortcode to any page.

4

Create Project

Click "New Project" and add a title, description, and instructions.

5

Start Chatting

Upload files and chat with AI about your documents!

Frequently Asked Questions

Do I need all 5 API keys?

No! You only need one API key to start chatting. OpenAI is required for project features (Vector Stores). Add more providers as needed.

Where are my API keys stored?

Your API keys are encrypted using AES-256-CBC and stored in your WordPress database. They never leave your server or touch our systems.

What file types can I upload?

PDF, DOC, DOCX, TXT, MD, CSV, JSON, XML, HTML, CSS, JS, PY, PHP. Default max size is 50MB per file (configurable up to 512MB).

Can multiple users access the same project?

Yes! In the Free version, there are 5 shared projects accessible to all logged-in users. Pro version adds per-user projects and sharing controls.

Can I use this on client sites?

Yes! The Free version is GPL licensed. For commercial use with advanced features like per-user projects and transcription, consider the Pro version.

How do I get support?

Free version: Use the WordPress.org support forum. Pro version: Priority email support at support@chatprojects.com.

What are the requirements?

WordPress 5.8 or higher, PHP 7.4 or higher, and at least one API key from OpenAI, Anthropic, Google, Chutes, or OpenRouter.
